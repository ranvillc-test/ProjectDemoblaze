Proyecto Demoblaze
05/08/2023

Proyecto de pruebas automatizado E2E sobre flujo de compra en Demoblaze.
El proyecto Demoblaze consta de la ejecución de 1 casos de pruebas, que cumple con la funcionalidad de finalizar el proceso de compra con dos productos en donde se seleccionan y se agregan al carrito los productos, posterior en el carrito de compra se listan los productos y se realiza el proceso de compra llenando un formulario con datos básicos del usuario y finalmente se validada la información y el estado exitoso de la compra

El proyecto se desarrolla bajo java 11   SDk versión 11

Tener en cuenta para la ejecución del proyecto:
- Importar este como proyecto maven
- Ejecución Maven para obtener el informe de ejecución
- La configuración de la ejecución se realiza con los comandos de RUN sobre RunnersFeature
Para el informe de pruebas se puede abrir por consola una vez ejecutados los casos
o también por medio de la carpeta target\site por medio del navegador
